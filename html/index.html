Wake County Public School System API

Given the remote learning changes required with COVID-19 the data provided here may not be 100% accurate until the start of the 2020-2021 school year

This site is designed to provide an API to give information about if school is in session or not for a given day. The data on this site is regenerated nightly at 12:05am ET. Below is a listing of active endpoints and their usage.

  • GET /api/today/ Gets the status of today
  • GET /api/tomorrow/ Gets the status of tomorrow
  • GET /api/thisweek/ Gets the status of this week
  • GET /api/nextweek/ Gets the status of next week
